## Getting Started

### Prérequis

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Cloner le repo
   ```sh
    git clone https://github.com/lucasconsejo/lucasconsejo.fr.git
   ```
2. Installation
   ```sh
    yarn install
    # ou
    npm install
   ```
3. Lancer le projet
   ```sh
    yarn dev
    # ou
    npm run dev
   ```
